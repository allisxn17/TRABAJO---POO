El proyecto consiste en el desarrollo de una aplicación que simula el funcionamiento de un ecosistema utilizando programación orientada a objetos en Python. En el ecosistema existen diferentes organismos, como plantas e insectos, que interactúan entre sí dentro de un entorno controlado por el sistema.

Las plantas pueden crecer, perder energía con el tiempo y aumentar su energía cuando ocurre un evento de lluvia. Los insectos pueden moverse dentro del ecosistema y perder energía durante su desplazamiento. Además, se pueden generar semillas que, después de un tiempo determinado, germinan y se convierten en nuevas plantas.

El sistema también incluye refugios dentro del ecosistema, los cuales permiten que los insectos recuperen energía cuando se encuentran en su misma posición. El ecosistema se encarga de gestionar todos los organismos y elementos presentes, actualizando su estado durante la simulación.
